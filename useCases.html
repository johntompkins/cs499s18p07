Use Cases

  1. Accessing the website and searching by name or keyword
    1. User navigates to the front page of the website
    2. The home page of the website shows, with a search bar and a list of recently gathered articles beneath it
    3. User types in search terms into the search box
    4. User optionally selects a start and end date for the search
    5. User hits the search button
    6. A list of articles is displayed beneath the search bar relevant to the typed name or keyword

  2. Downloading search results as a comma seperated values file
    1. User follows the steps above to search for a keyword or name
    2. User clicks the download results button, located at the top right of the search results
    3. A download begins, allowing the user to select where to save the .csv file
    4. The user downloads the .csv file and can open it in Excel, notepad, or any equivalent program
    5. The file contains the names of articles returned by the search, in .csv format

  3. Reading an article and possibly going to the source website
    1. Optionally, user follows the steps above to search for a keyword or name
    2. User selects an article from the front page or returned search results if they conducted a search
    3. User is taken to a specific page for the article
    4. The page contains the article information, including Title, Authors, Publish Date, Article Text, and more
    5. User browses information, possible scrolling down to look at related images, if any, at the bottom of the article
    6. If the user wishes, they may click the link to original on the left sidebar
    7. User will be taken to the original website if they click the link
